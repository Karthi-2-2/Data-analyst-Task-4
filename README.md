# SQL Data Analysis Task 4

This project was completed as part of a Data Analyst Internship Task. The main goal was to practice SQL queries and understand how data can be analyzed using different SQL operations.

For this task, I used the Train dataset and performed various SQL queries such as filtering records, sorting data, calculating averages, grouping data, creating views, and indexing tables. I also explored the use of subqueries and joins to retrieve meaningful information from the database.

### Tasks Performed

* Displayed all records from the dataset
* Filtered records based on cost values
* Sorted data using ORDER BY
* Calculated average cost using aggregate functions
* Used subqueries to find records above the average cost
* Grouped shipment data and counted total orders
* Created a view for shipment summary analysis
* Created an index to improve query performance
* Practiced INNER JOIN operations
